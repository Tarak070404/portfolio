chapter-1

starting with the basics


what is HTML:

  1. HTML stands for Hyper Text Markup Language
  2. HTML is the standard markup language for creating Web pages
  3. HTML describes the structure of a Web page
  4. HTML consists of a series of elements

what is css:

  1. CSS is used for styling web pages and controlling the presentation of HTML documents
  2. It allows developers to separate the structure (HTML) and the style (CSS) of a web page.
  3. CSS properties define the style aspects (e.g., color, font-size, margin) of an element.
  4. Values are assigned to properties to specify the desired style.

what is js:

  1. JavaScript is a client-side scripting language, executed in the user's browser, enabling dynamic and interactive web page behavior.
  2. JavaScript interacts with the Document Object Model (events), allowing developers to dynamically update and modify HTML content and structure.
  3. JavaScript is used to respond to user actions (events) like clicks and keypresses through event listeners, enabling interactive and responsive web applications.
  4. JavaScript supports asynchronous programming, crucial for tasks like fetching data from servers without blocking the user interface, using mechanisms like Promises and async/await.

what is typescript:

  1. TypeScript is a superset of JavaScript, meaning that all existing JavaScript code is valid TypeScript. Developers can gradually adopt TypeScript in their projects
  2. TypeScript introduces static typing, allowing developers to define and enforce types for variables, function parameters, and return values.
  3. TypeScript supports object-oriented programming concepts like classes, interfaces, inheritance, and access modifiers.
  4. TypeScript code is transpiled to JavaScript before execution, and during this process, the TypeScript compiler performs static type checkin
